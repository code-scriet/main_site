## 🧠 About

**codescriet**  is the official coding club of [Sir Chhotu Ram Institute of Engineering and Technology (SCRIET)](https://scriet.ac.in/), a constituent college of **Chaudhary Charan Singh University (CCSU), Meerut, Uttar Pradesh, India**.

Founded by undergraduate students, the club provides a structured space to learn Data Structures & Algorithms, build real-world projects, prepare for placements, and connect with alumni in the software industry.

> 🌐 This very website is a **student-built project** — deployed on Render, fronted by Cloudflare, and shipped through the codescriet GitHub organisation.

## 🚀 Getting Started (Local Setup)

### Prerequisites

- Node.js `v18+`
- npm or yarn
- Git

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/code-scriet/main_site.git
cd main_site

# 2. Install dependencies
npm install

# 3. Set up environment variables
cp .env.example .env
# Fill in the required values in .env

# 4. Run database migrations
npx prisma migrate dev

# 5. Start the development server
npm run dev
```

The app will be running at `http://localhost:3000`

## 🤝 Contributing

We welcome contributions from all students of CCSU and beyond!

### Steps to Contribute

```bash
# 1. Fork this repository
# 2. Clone your fork
git clone https://github.com/YOUR_USERNAME/main_site.git
cd main_site

# 3. Add upstream remote
git remote add upstream https://github.com/code-scriet/main_site.git

# 4. Create a new branch (NEVER push to main directly)
git checkout -b feat/your-feature-name

# 5. Make your changes, then stage and commit
git add .
git commit -m "feat: describe your change clearly"

# 6. Push your branch
git push origin feat/your-feature-name

# 7. Open a Pull Request on GitHub
```

### Contribution Guidelines

- Always work on a **new branch**, never directly on `main`
- Write **clear, descriptive commit messages**
- Keep PRs focused — one feature or fix per PR
- Follow the existing **code style and folder structure**
- Test your changes locally before submitting
