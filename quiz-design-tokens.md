| Pattern | Config |

| **Panel switch** | `initial={{ opacity: 0, x: -20 }} animate={{ opacity: 1, x: 0 }} exit={{ opacity: 0, x: 20 }} transition={{ duration: 0.3 }}` |

## 6. Special Effects

### Glassmorphism
- **Light glass**: `bg-white/70 backdrop-blur-xl border border-white/20 shadow-xl` (`.glass`)
- **Dark glass**: `bg-black/30 backdrop-blur-xl border border-white/10` (`.glass-dark`)
- **Header**: `bg-white/95 backdrop-blur supports-[backdrop-filter]:bg-white/80`
- **Sign-in card**: `backdrop-blur-sm bg-white/90`

### Gradient Text
- `.gradient-text`: `bg-clip-text text-transparent bg-gradient-to-r from-amber-500 to-orange-500`

### Grain/Noise Texture
- `.grain-overlay::after`: SVG noise at 3.5% opacity
- `.noise-bg`: SVG noise at 4% opacity
- Used for premium surface treatment

### Performance
- `.performance-surface`: `transform: translateZ(0); backface-visibility: hidden; will-change: transform, opacity;`
- `.gpu-layer`: Same GPU promotion for scroll animations
- Mobile: `contain: layout style paint;` on sections, reduced backdrop blur
